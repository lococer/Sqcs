# 关于控制算法：

## 大纲

定义了电机效率为`[0-100]`%  
定义了电机效率表`channelPulse[]`，存放并**执行**了每个电机当前的效率，初始值均为0%   
开辟了两个线程分别用于<u>实时**执行**电机效率</u>和<u>实时**更新**PCA9685的PWM信号</u>
